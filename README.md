# 💸 Split Bill

SwiftUI ile geliştirilmiş, şık ve pratik bir hesap/bahşiş bölüştürme uygulaması. Arkadaş grupları için ödeme sürecini saniyeler içinde çözer.

## ✨ Öne Çıkan Özellikler
- **Hızlı Hesaplama:** Toplam tutar, bahşiş yüzdesi ve kişi sayısına göre anında sonuç.
- **Güvenli Altyapı:** Sıfıra bölme (0 kişi girilmesi) çökmelerini engelleyen özel hata yönetimi.
- **Premium UX:** Animasyonlu açılış ekranı (Splash), özel fontlar ve akıllı klavye yönetimi.

## 🛠 Teknoloji Yığını
- **UI:** %100 Native SwiftUI
- **Mimari:** MVVM (Model-View-ViewModel)
- **Test:** XCTest ile TDD (Test-Driven Development) uyumlu Unit Testler.
